CRUD Operations with MongoDB

CRUD is an acronym that stands for Create, Read, Update, and Delete. These are the basic operations performed in a database or a data-driven application. In this example, we will explain how CRUD operations work with MongoDB, a NoSQL database.

1. Create

The Create operation refers to the action of adding new data (document) to a MongoDB collection. This is done using the insertOne() or insertMany() method in MongoDB.

  • Example: Adding a new user document to a topic collection.
  • db.topic.insertOne({ name: 'Frontend', description: 'web GUI' })

2. Read

The Read operation refers to retrieving data (documents) from a MongoDB collection. This is achieved using the find() or findOne() method.

  • Example: Fetching user details from the topic collection.
  • db.topics.findOne({ _id: ObjectId('60c72b2f9b1d8a5e7c2a1234') })

3. Update

The Update operation is used to modify existing documents in a collection. This is done using the updateOne() or updateMany() method.

  • Example: Updating a topic's description in the topic collection.
  • db.topic.updateOne({ _id: ObjectId('60c72b2f9b1d8a5e7c2a1234') }, { $set: { description: 'user interface in a web browser' } })

4. Delete

The Delete operation refers to removing documents from a MongoDB collection. This is done using the deleteOne() or deleteMany() method.

  • Example: Deleting a topic document from the topic collection.
  • db.topic.deleteOne({ _id: ObjectId('60c72b2f9b1d8a5e7c2a1234') })
